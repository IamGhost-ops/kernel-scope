## Guidelines for Contributing

- You can choose an issue from the issues tab or raise a new issue yourself.
- The assignment of issue is completely first come first serve.
- If you want to work on an issue first leave a comment below it (e.g., "I would like to work on this").
- Do NOT start working on the issue until a maintainer formally assigns the issue to you.
- Unassigned Pull Request (PRs) related to the issue will not be accepted to avoid duplicated work.
- After an issue is assigned it should be completed within 3 days or it will be reassigned to someone else
