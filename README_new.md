# Tetragon

Tetragon is a system to observe and analyze Linux system behavior layered security mechanisms.

## Phase 1: Full System Tracing
The system captures all system calls without filtering to establish a complete behavioral baseline.

## Phase 2: File integrity Monitoring (FIM)
(comming soon) Detection of changes based on the baseline behavior.

## Phase 3: LSM-based Policy Enforcement
(comming soon) Security rules applied at kernel level.

## Goal
To built a layered model for observing, analyzing, and controlling system behavior over time.
